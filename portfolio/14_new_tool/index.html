Map with a new tool

Whitebox Tools: The catchment of the River Kemp, South Shropshire

Tools I used

  • Whitebox Tools
  • QGIS

What did I learn?

  • I used Whitebox Tools via the python whitebox tools runner interface to generate watershed and hypsometric tinting. This is my new tool.
  • Whitebox Tools has very limited support for coordinate reference systems at the moment.
  • Using shapeburst styled polygons to highlight watershed in a QGIS layout (from van der Kwast hydrology book).
  • That if you have two projects open in QGIS you can copy and paste a group of layers from one to the other, fully styled.
